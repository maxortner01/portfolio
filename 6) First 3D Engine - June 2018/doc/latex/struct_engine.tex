\subsection{Detailed Description}
Controls high level flow of engine. 

The \doxyref{Engine}{p.}{struct_engine} struct handles the creation and destruction of the game process. It is is meant to be inherited by the main process in the \char`\"{}main\char`\"{} folder. Before loop can be called a Screen object must be set.
